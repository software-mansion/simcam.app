# expo-camera patches

This folder contains `patch-package` patches for `expo-camera` versions that still block simulator camera usage at compile time.

## Available patches

- `17.0.8`
- `17.0.9`
- `17.0.10`
- `55.0.8`
- `55.0.9`
- `55.0.10`

## Important version note

`expo-camera` has the upstream fix as of `55.0.11` (PR: [expo/expo#44159](https://github.com/expo/expo/pull/44159)).

For `55.0.11` and newer, no patch from this folder is required.
