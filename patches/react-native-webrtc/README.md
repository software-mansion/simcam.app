# react-native-webrtc patches

This folder contains `patch-package` patches for `react-native-webrtc` versions that perform simulator checks in a way that blocks runtime camera devices.

## Available patches

- `106.0.7`
- `111.0.6`
- `118.0.7`
- `124.0.0`
- `124.0.4`
- `124.0.7`

Upstream reference: [react-native-webrtc#1801](https://github.com/react-native-webrtc/react-native-webrtc/pull/1801)
