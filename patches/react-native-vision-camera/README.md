# react-native-vision-camera patches

This folder contains `patch-package` patches for `react-native-vision-camera` `4.x` versions that disable camera behavior on simulator builds.

## Available patches

- `4.5.0`
- `4.5.3`
- `4.6.0`
- `4.6.4`
- `4.7.0`
- `4.7.3`

## Important version note

VisionCamera `5.x` does not require patches from this folder.

Upstream reference: [react-native-vision-camera#3724](https://github.com/mrousavy/react-native-vision-camera/pull/3724)
